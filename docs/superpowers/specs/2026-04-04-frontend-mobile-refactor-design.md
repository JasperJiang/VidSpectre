# VidSpectre 前端移动端重构设计方案

## 概述

将 VidSpectre 前端从 Bootstrap 5 迁移到 Tailwind CSS，实现移动端优先的响应式设计，提升移动设备上的用户体验。

## 技术选型

- **CSS 框架**: Tailwind CSS v3 (Play CDN)
- **JavaScript**: 提取到 `static/js/app.js`，使用 `addEventListener`
- **主题**: 深色主题（保持一致）
- **构建方式**: 无需构建，Play CDN 直接引入

## 页面改动

### 1. 订阅列表页 (index.html)

#### 桌面端
- 卡片列表，每张卡片显示一个订阅
- 卡片内：名称、类型、当前集数、关键字、最新更新
- 操作按钮：🔄 爬取（主操作）、⋯ 更多（展开菜单含保存/编辑/删除）
- 底部周期选择器单独一行

#### 移动端 (< 768px)
- 全宽卡片堆叠
- 紧凑布局
- 保持相同功能

#### 卡片结构

```
┌─────────────────────────────────────────┐
│ 🎬 电视剧名称                    [展开] │
│ 看到: 第10集  关键字: HDR              │
│ 最新: 第24集                           │
├─────────────────────────────────────────┤
│  [🔄 爬取]        [⋯ 更多 ▾]          │
└─────────────────────────────────────────┘

「⋯ 更多」菜单:
├── 💾 保存
├── ✏️ 编辑
└── 🗑️ 删除
```

### 2. 剧集展开列表

#### 桌面端
- 3列网格布局
- 每集一行，显示所有链接

#### 移动端
- 单列堆叠
- 每集：集数标题 + 「获取」按钮
- 点击「获取」→ 显示 "获取中..." → prompt 复制磁力链接
- 周期选择器单独一个区块在底部

```
┌─────────────────────────────────────────┐
│ 第24集                           [获取] │
│ ─────────────────────────────────────── │
│ 第23集                           [获取] │
│ ─────────────────────────────────────── │
│ 第22集                           [获取] │
│ ...                                     │
├─────────────────────────────────────────┤
│ 周期: [每6小时 ▼]                      │
└─────────────────────────────────────────┘
```

### 3. 添加订阅页 (subscription.html)
- Tailwind 卡片式表单
- 搜索结果使用 Tailwind list-group 样式

### 4. 编辑订阅页 (edit_subscription.html)
- Tailwind 表单卡片

### 5. 设置页 (settings.html)
- Tailwind 表单卡片
- 预设选择器联动

### 6. 基础模板 (base.html)
- 引入 Tailwind CSS Play CDN
- 引入 `static/js/app.js`
- 移除内联 onclick 处理
- 深色主题通过 Tailwind `dark:` 变体实现

## JavaScript 重构

### 文件结构
```
static/
└── js/
    └── app.js
```

### app.js 函数

| 函数 | 作用 |
|------|------|
| `initSaveButtons()` | 绑定「保存」按钮点击 |
| `initFetchButtons()` | 绑定「爬取」按钮点击 |
| `initEpisodeToggles()` | 绑定「展开/收起」按钮 |
| `initIntervalSelectors()` | 绑定周期选择器联动 |
| `initDeleteForms()` | 绑定删除确认 |
| `initSearchKeyword()` | 绑定搜索回车事件 |
| `initMoreMenus()` | 绑定「更多」下拉菜单 |

### 全局函数（暴露在 window 上，供 HTML 调用）
- `fetchSubscription(btn)` — 爬取订阅
- `saveSubscription(subId)` — 保存订阅
- `toggleEpisodes(btn)` — 展开/收起剧集

## Tailwind 配置

### 深色主题
使用 `class` strategy，手动在 html 标签添加 `dark` class：
```html
<html class="dark">
```

### 自定义颜色（可选）
保持现有的深色配色，不额外定义。

## 迁移步骤

1. 创建 `static/js/app.js`
2. 重写 `base.html` — 引入 Tailwind + JS 文件
3. 重写 `index.html` — 卡片布局 + Tailwind
4. 重写 `subscription.html` — Tailwind 表单
5. 重写 `edit_subscription.html` — Tailwind 表单
6. 重写 `settings.html` — Tailwind 表单
7. 实现 `app.js` — 所有事件绑定
8. 测试所有功能

## 风险与注意事项

1. **Bootstrap 依赖移除** — 需确认没有其他第三方依赖
2. **内联 JS 迁移** — 确保所有 onclick 都迁移到 addEventListener
3. **深色主题一致性** — 确保所有页面在深色模式下正常显示
4. **API 兼容性** — 不改变任何 API 接口，只改前端
